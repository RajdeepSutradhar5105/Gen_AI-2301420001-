## Overview
This repository contains a comprehensive collection of Python-based experiments focused on probability distributions, statistical estimation, and the training of various neural network architectures, including Generative Adversarial Networks (GANs) and Variational Autoencoders (VAEs).

## Experiments Roadmap

### 1. Probability Distributions
* **File:** `Experiment_1.ipynb`
* **Description:** Generates and visualizes random samples from key probability distributions (Normal, Uniform, Binomial, and Poisson) using NumPy, Matplotlib, and Seaborn.

### 2. Maximum Likelihood Estimation (MLE)
* **File:** `Experiment_2.ipynb`
* **Description:** Implements the MLE method to estimate distribution parameters from synthetic data using numerical optimization via `scipy.optimize.minimize`.[cite: 2]

### 3. Neural Network Training (MNIST)
* **File:** `Experiment_3.ipynb`
* **Description:** Trains a feedforward neural network on the MNIST benchmark dataset using TensorFlow/Keras and Stochastic Gradient Descent (SGD).

### 4. Generative Adversarial Networks (GAN)
* **File:** `Experiment_4.ipynb`
* **Description:** Trains a GAN on the MNIST dataset using PyTorch, monitoring the loss of both the generator and discriminator.

### 5. Flow-Based Generative Models
* **File:** `Experiment_5.ipynb`
* **Description:** Implements a flow-based generative model (RealNVP) using coupling layers and deep learning frameworks.

### 6. Evaluation Metrics
* **File:** `Experiment_6.ipynb`
* **Description:** Implements core classification evaluation metrics, including Accuracy, Precision, Recall, F1-Score, and the Confusion Matrix using `scikit-learn`.

### 7. Performance Evaluation
* **File:** `Experiment_7.ipynb`
* **Description:** Evaluates trained model performance metrics on a separate validation set to monitor loss and accuracy.

### 8. Variational Autoencoder (VAE)
* **File:** `Experiment_8.ipynb`
* **Description:** Builds a VAE architecture consisting of an encoder and a decoder for the FashionMNIST dataset.

### 9. Autoregressive Models
* **File:** `Experiment_9.ipynb`
* **Description:** Implements an autoregressive model using an LSTM network to predict and generate sequence patterns.

### 10. GAN Loss Analysis
* **File:** `Experiment_10.ipynb`
* **Description:** Analyzes various loss functions used in GAN training, such as adversarial loss, feature matching loss, and Wasserstein loss.
